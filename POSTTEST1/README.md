# Sistem Penjualan Perhiasan 💎

## Deskripsi Program

Program **Sistem Penjualan Perhiasan** merupakan aplikasi berbasis Java yang dibuat untuk mengelola data perhiasan dan transaksi penjualan. Program ini menggunakan konsep **Pemrograman Berorientasi Objek (PBO)** seperti **Class, Object, Constructor, dan ArrayList**.

Program ini berjalan pada **console** dan memiliki fitur untuk menambah, menampilkan, mengupdate, menghapus data perhiasan serta melakukan transaksi penjualan.

---

## Fitur Program

Program ini memiliki beberapa fitur utama yaitu:

1. **Tambah Data Perhiasan**
   Digunakan untuk menambahkan data perhiasan baru.

2. **Lihat Data Perhiasan**
   Menampilkan seluruh data perhiasan yang tersimpan.

3. **Update Data Perhiasan**
   Mengubah data perhiasan yang sudah ada.

4. **Hapus Data Perhiasan**
   Menghapus data perhiasan berdasarkan ID.

5. **Transaksi Penjualan**
   Digunakan untuk mencatat transaksi pembelian perhiasan oleh pelanggan.

6. **Lihat Data Transaksi**
   Menampilkan seluruh data transaksi yang telah dilakukan.

7. **Keluar Program**
   Mengakhiri program.

---

## Struktur Program

Program ini terdiri dari beberapa class utama:

### 1. Main

Class utama yang digunakan untuk menjalankan program serta menampilkan menu program.

### 2. Perhiasan

Class yang digunakan untuk menyimpan data perhiasan seperti:

* ID Perhiasan
* Nama Perhiasan
* Jenis Perhiasan
* Harga
* Stok

### 3. Transaksi

Class yang digunakan untuk menyimpan data transaksi seperti:

* ID Transaksi
* Nama Pembeli
* Nama Perhiasan
* Jumlah Pembelian
* Total Harga
